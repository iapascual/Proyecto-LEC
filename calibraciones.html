Magnitudes dimensionales

Nuestra acreditación nos habilita para realizar calibraciones de magnitudes dimensionales de cintas métricas, reglas, calibres y micrómetros.

Magnitudes eléctricas

Nuestra acreditación nos habilita para realizar calibraciones de una gran variedad de instrumentos de medición de magnitudes eléctricas. Entre ellos se encuentran multímetros, ohmétros, analizadores de potencia y frecuencímetros.

Nuestro laboratorio forma parte de la red SAC para la realización de estas calibraciones.

Temperatura

Nuestra acreditación nos habilita para realizar calibraciones de distintos tipos de termómetros y termosondas.

Entre los termómetros se pueden contar termómetros de alcohol, de mercurio, digitales y digitales con sensor externo.

Entre las termosondas se pueden contar termocuplas de distinto tipo, sensores resistivos (PT-100 y similares) y semiconductores.
